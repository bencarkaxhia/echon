# 🚀 CLOUDFLARE R2 STORAGE SETUP GUIDE

**Complete guide to enable cloud storage for Railway deployment**

---

## 📦 STEP 1: CREATE CLOUDFLARE R2 ACCOUNT

1. Go to: https://dash.cloudflare.com/sign-up
2. Sign up (100% FREE - 10GB storage forever!)
3. Verify your email

---

## 🪣 STEP 2: CREATE R2 BUCKET

1. In Cloudflare Dashboard → Click **R2**
2. Click **Create Bucket**
3. Bucket name: `echon-storage`
4. Location: **Automatic** (best performance)
5. Click **Create Bucket**

---

## 🔑 STEP 3: GET API CREDENTIALS

1. In R2 section → Click **Manage R2 API Tokens**
2. Click **Create API Token**
3. Token name: `echon-backend`
4. Permissions: **Object Read & Write**
5. TTL: **Forever**
6. Click **Create API Token**

**COPY THESE 3 VALUES** (you'll need them!):
```
Access Key ID: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
Secret Access Key: yyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyyy
Endpoint: https://xxxxxxxxxxxxx.r2.cloudflarestorage.com
```

---

## 🚂 STEP 4: ADD TO RAILWAY

1. Go to your Railway project: https://railway.app/project/your-project
2. Click on your **backend service**
3. Go to **Variables** tab
4. Add these environment variables:

```bash
R2_ENDPOINT=https://xxxxxxxxxxxxx.r2.cloudflarestorage.com
R2_ACCESS_KEY=your_access_key_id_here
R2_SECRET_KEY=your_secret_access_key_here
R2_BUCKET_NAME=echon-storage
```

5. Click **Deploy** (Railway will restart with new config)

---

## 📦 STEP 5: INSTALL BOTO3

Add to `requirements.txt`:
```
boto3==1.34.34
```

Then redeploy!

---

2. Login
3. Upload a photo
4. **IT WORKS!** Photo saved to Cloudflare R2!
