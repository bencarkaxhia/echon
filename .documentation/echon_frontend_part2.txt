## 🔧 TROUBLESHOOTING

### Issue: "Network Error" when creating space
**Fix:** Make sure backend is running on port 8000

### Issue: Frontend can't reach backend
**Fix:** Check `.env` file has `VITE_API_URL=http://localhost:8000`

### Issue: Protected route redirects to login even after registering
**Fix:** Check browser console for token storage issues

### Issue: Doors still overlapping on desktop
**Fix:** Clear browser cache and hard refresh (Ctrl+Shift+R)
