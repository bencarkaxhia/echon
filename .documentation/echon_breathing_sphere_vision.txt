## 💻 CODE SNIPPETS:

### Breathing Animation:
```typescript
// In RotatingSphere.tsx
<motion.div
  animate={{
    scale: [1, 1.05, 1],
    opacity: [0.8, 1, 0.8]
  }}
  transition={{
    duration: 4,
    repeat: Infinity,
    ease: "easeInOut"
  }}
>
```

### Sphere Tilt (React to door hover):
```typescript
// In DoorScene.tsx
<RotatingSphere
  tiltToward={hoveredDoor} // "memories", "family", etc.
/>

// In RotatingSphere:
const tiltAngles = {
  memories: { x: -10, y: -10 },  // Top-left
  stories: { x: -10, y: 10 },    // Top-right
  family: { x: 10, y: -10 },     // Bottom-left
  now: { x: 10, y: 10 }          // Bottom-right
};

<motion.div
  animate={{
    rotateX: tiltToward ? tiltAngles[tiltToward].x : 0,
    rotateY: tiltToward ? tiltAngles[tiltToward].y : 0
  }}
>
```

### Energy Rays:
```typescript
const rays = [
  { door: 'memories', angle: -135, length: 300 },
  { door: 'stories', angle: -45, length: 300 },
  { door: 'family', angle: 135, length: 300 },
  { door: 'now', angle: 45, length: 300 }
];

{rays.map(ray => (
  <motion.div
    key={ray.door}
    className="absolute origin-center"
    style={{
      width: '2px',
      height: `${ray.length}px`,
      background: `linear-gradient(to top, 
        rgba(212, 175, 55, 0), 
        rgba(212, 175, 55, ${hoveredDoor === ray.door ? 0.6 : 0.2})
      )`,
      transform: `rotate(${ray.angle}deg)`,
      transformOrigin: 'bottom center'
    }}
    animate={{
      opacity: hoveredDoor === ray.door ? 1 : 0.3
    }}
  />
))}
```
